Eine Auswahl meiner Arbeiten

OBA Berufswünsche

Mit dieser Web Applikation ist es möglich, Wünsche für die Berufswahl in Form von kurzen Texten zu teilen. Neue Beiträge von Jugendlichen und Erwachsenen erscheinen in Echtzeit auf der Wunschliste.

Die Applikation kam an der Ostschweizer Bildungs-Ausstellung (OBA) 2012 zum Einsatz.

Technologien: HTML5, CSS3, PHP (Silex Microframework), SQLite

IT Lernende Social Network

Dieses Enterprise Social Network steht exklusiv den IT-Lernenden der Leica Geosystems AG zur Verfügung. Es ermöglicht ihnen eine effiziente und abteilungsübergreifende Kommunikation.

Der Activity Stream mit Status Updates und Kommentaren stellt die Kernfunktion dar. Weitere Funktionen sind Benutzerprofile, Gruppen und private Nachrichten.

Eine dazugehörige Webseite zeigt ausgewählte Informationen aus den Profilen, wie zum Beispiel die Telefonnummer oder die aktuelle Abteilung, an.

Technologien: HTML5, CSS3, PHP (WordPress und BuddyPress), MySQL

LinkShuffler

LinkShuffler ist ein einfacher und blitzschneller Bookmark Manager. Listen erleichtern die Verwaltung der Linksammlung. Innerhalb dieser selbst erstellten Listen können die Links sortieren werden. Die Suchfunktion hilft beim schnellen Wiederauffinden der einzelnen Links. Medien von Youtube, Flickr oder Soundcloud werden mit Hilfe der oEmbed Schnittstelle automatisch angezeigt.

Technologien: HTML5, CSS3, PHP (Codeigniter), MySQL
